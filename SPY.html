1. Setting Up the Environment

We begin by importing the necessary packages:

  • Pandas is the go-to package for data manipulation.
  • NumPy provides support for mathematical functions.
  • Matplotlib and Seaborn are used for data visualization.
  • Glob helps us retrieve files with specific patterns.

We then create a list of the companies we want to analyze. For this project, we focus on four major tech stocks: Apple, Amazon, Google, and Microsoft.

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
import glob
